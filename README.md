## 🛠️ Tecnologias e Ferramentas Utilizadas

Para a execução bem-sucedida desta atividade, foram empregadas as seguintes tecnologias:

- **Sistema Operacional:** Windows 10/11.
- **Servidor Web:** Apache HTTP Server (versão 2.4).
- **Aplicação:** Jogo 2048 (Código-fonte estático via GitHub).
- **Documentação Técnica:** Markdown para estruturação do relatório.

---

## 🎯 Objetivos da Atividade

O foco central deste projeto foi aplicar conceitos de **Quality Assurance (QA)** e gerenciamento de configuração:

1.  **Deploy Manual:** Execução passo a passo da configuração do servidor sem automação, garantindo o entendimento total do fluxo.
2.  **Configuração de DocumentRoot:** Manipulação do arquivo `httpd.conf` para direcionamento de arquivos no Apache.
3.  **Verificação de Ambiente:** Testes de acesso via `localhost` para confirmar a integridade da aplicação após o deploy.
4.  **Padronização de Processos:** Criação de documentação clara e replicável, essencial para a manutenção de software.
